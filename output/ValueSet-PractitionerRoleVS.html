ValueSet: PractitionerRole Code

Official URL: http://hl7.org/fhir/us/fhir-directory-query/ValueSet/PractitionerRoleVS Version: 0.0.1
Active as of 2022-07-13 Computable Name: PractitionerRoleVS

Codes for the capabilities that an individual, group, or organization is acknowledged to have in a payer network, including general codes from the HL7 PractitionerRole Code System.

References

  • National Directory Endpoint Qry Exchange PractitionerRole

Logical Definition (CLD)

This value set includes codes based on the following rules:

  • Include all codes defined in http://hl7.org/fhir/us/fhir-directory-query/CodeSystem/ProviderRoleCS
  • Include all codes defined in http://terminology.hl7.org/CodeSystem/practitioner-role
