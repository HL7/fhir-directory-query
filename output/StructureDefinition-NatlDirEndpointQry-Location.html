Resource Profile: National Directory Endpoint Qry Exchange Location

Official URL: http://hl7.org/fhir/us/fhir-directory-query/StructureDefinition/NatlDirEndpointQry-Location Version: 0.0.1
Active as of 2022-07-13 Computable Name: NatlDirEndpointQryLocation

A Location is the physical place where healthcare services are provided, practitioners are employed, organizations are based, etc. Locations can range in scope from a room in a building to a geographic region/area.

Usage:

  • Refer to this Resource Profile:
  • Examples for this Resource Profile: Cancer Clinic, OrgA CT Location 1, Hartford Hospital Location 1, Hartford Hospital Location 2, OrgA CT Location 1, OrgA CT Location 2, OrgA MA Location 1, OrgA MA Location 2, State of CT Area
